---
title: "API Configuration"
---

The API is only configurable in the [Enterprise version](/docs/enterprise) of BuildBuddy. For more information, view the [API Documentation](/docs/enterprise-api).

## Section[](#section "Direct link to Section")

`api:` The API section enables the BuildBuddy API over both gRPC(s) and REST HTTP(s). **Optional**

## Options[](#options "Direct link to Options")

**Optional**

* `enable_api:` Whether or not to enable the BuildBuddy API.
* `api_key:` The default API key to use for on-prem enterprise deploys with a single organization/group.

## Example section[](#example-section "Direct link to Example section")

```
api:
  enable_api: true

```
